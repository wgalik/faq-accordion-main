# FAQ Accordion Component

A responsive FAQ accordion component built as part of a Frontend Mentor challenge. Users can expand and collapse questions to reveal the answers. The project is designed using HTML and SCSS with PostCSS for styling.

## Demo

[Site](https://wgalik.github.io/faq-accordion-main/)

## Technologies

- **HTML5** – structured semantic markup for questions and answers
- **SCSS + PostCSS** – styling with variables, nesting, and responsive design
- **CSS Transitions** – smooth animation for expanding and collapsing answers

## Features

- Expand/collapse FAQ items by clicking on the question
- Only one FAQ item is open at a time using radio buttons
- Smooth transition effect for content reveal
- Fully responsive layout for mobile and desktop screens
- Clean, minimal, and accessible design

## Usage

- Click on a question to expand or collapse the answer.
- Only one question can be open at a time.
- The plus/minus icon changes dynamically based on the state of the question.

## Learning Points

- SCSS practice: variables, nesting, responsive styling
- CSS transitions for smooth animations
- Structuring semantic HTML for accessibility
- Using radio buttons to manage accordion state without JavaScript
